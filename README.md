# 网易我的世界模组自动注入器 (Netease Auto Injector)

[![Python](https://img.shields.io/badge/Python-3.6+-blue.svg)](https://www.python.org/)

这是一个专为**网易版我的世界 (Minecraft China)** 设计的自动化辅助脚本。它解决了网易启动器在本地联机模式下强制清空/重置 `mods` 文件夹，导致无法加载自定义模组（Mod）的问题。

此外，本脚本还支持**自动注入光影包和材质包**，让您在网易版中轻松使用高清资源。

---

## 📂 目录结构说明

下载并解压本项目后，您需要关注以下几个文件夹（脚本首次运行会自动创建）：

| 文件夹名称 | 用途 | 放入文件类型 |
| :--- | :--- | :--- |
| **`MyMods`** | 存放您的模组文件 | `.jar` |
| **`MyResourcePacks`** | 存放材质包/资源包 | `.zip` 或 `.mcpack` |
| **`MyShaderPacks`** | 存放光影包 (Shader) | `.zip` |

**脚本文件**：`NeteaseModInjector.py` (即本项目的主程序)

---

## 🚀 使用方法

### 1. 准备工作
1.  确保您已安装 Python 3 (勾选 "Add Python to PATH")。
2.  将您的 `.jar` 模组文件放入 `MyMods` 文件夹。
3.  (可选) 将光影包放入 `MyShaderPacks`，材质包放入 `MyResourcePacks`。

### 2. 启动脚本
**在启动游戏之前**，请先运行本脚本。
- 双击运行脚本，或者在终端输入：
  ```bash
  python NeteaseModInjector.py
  ```
- 脚本启动后会显示“正在监控游戏启动...”，此时**请不要关闭黑色窗口**。

### 3. 启动游戏
1.  打开网易我的世界启动器。
2.  进入单人游戏或本地联机房间。
3.  **关键时刻**：当启动器开始加载游戏（清理 Mods 文件夹）的那一瞬间，脚本会自动检测到并**极速注入**您准备好的模组。
4.  您会在脚本窗口看到 `⚡ 模组注入完成！` 的提示。
5.  游戏加载完成后，您的模组应当已经成功加载。

## ❓ 常见问题 (FAQ)

### Q1: 我需要手动输入游戏的 Mods 路径吗？
**不需要。**
脚本内置了智能检测功能，会自动读取 Windows 注册表 (`HKEY_CURRENT_USER\Software\Netease\MCLauncher`) 来获取您的网易我的世界安装路径。您只需要运行脚本即可。

### Q2: 光影和材质包是怎么替换的？
脚本会在启动时（以及每次模组注入时）检查 `MyShaderPacks` 和 `MyResourcePacks` 文件夹。如果检测到里面的文件在游戏中不存在，它会自动复制进去。
*注意：这不会覆盖游戏内同名的现有文件，只会补充缺失的文件。*

### Q3: 为什么脚本要一直开着？
因为网易启动器可能会在游戏启动过程中的不同阶段（甚至多次）清空文件夹。脚本保持运行可以进行毫秒级监控 (10ms 轮询)，确保在文件被删除后的瞬间立即补回去，保证 100% 的成功率。
